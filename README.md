##  Overview
The **Campus Navigation Application** is a Python-based system that helps students and visitors navigate a university campus by finding optimal routes between buildings. The application uses graph-based pathfinding algorithms, supports accessibility-aware routing, and includes searching and data-loading features.

This project follows Agile methodology with multiple sprints, user stories, branch-based development, and comprehensive testing (black-box, white-box, symbolic execution, and concolic testing).

---

##  Features
- Load campus map data from file  
- Shortest path calculation using **Dijkstra's algorithm**  
- Optional **accessible-friendly routes**  
- Building & POI search  
- Estimated travel time calculation  
- Save/Load user favourites  
- CLI-based user interface  
- Full test suite (black-box & white-box)  
- Symbolic and concolic testing on core algorithms
